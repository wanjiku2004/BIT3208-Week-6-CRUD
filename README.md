This repository contains EduStream, a simple, one-page Student Management System built using a LAMPP (XAMPP for Linux) environment. The system features a modern frontend web dashboard (index.php) where users can fill out a registration form to dynamically save student profiles (Names, Emails, and GPAs) without reloading the page, a secure PHP API script (add_student.php) that uses prepared statements to safely pass inputs and prevent SQL injection, and a structured MySQL database dump file (database.sql) to automatically instantiate the required table schemas locally. To launch the application, clone this repository directly inside your system's /opt/lampp/htdocs/ folder, import the database.sql file via phpMyAdmin under a new database named school_system, ensure your Apache and MySQL servers are running, and navigate to http://localhost/school-system/index.php in your web browser.
