This repository contains a simple Student Management System built using a LAMPP (XAMPP for Linux) environment. The system features a modern frontend web dashboard (index.php) where users can fill out a registration form to dynamically save student profiles (Names, Emails, and GPAs) without reloading the page, a secure PHP API script (add_student.php) that uses prepared statements to safely pass inputs and prevent SQL injection, and a structured MySQL database dump file (database.sql) to automatically instantiate the required table schemas locally. To launch the application, clone this repository directly inside your system's /opt/lampp/htdocs/ folder, import the database.sql file via phpMyAdmin under a new database named school_system, ensure your Apache and MySQL servers are running, and navigate to http://localhost/school-system/index.php in your web browser.

The Library Management System is a web-based application developed using PHP and MySQL to facilitate the management of library records. The system allows users to store, manage and retrieve book information including Book ID, Book Title, Author, and Category. It provides a simple interface for adding new books to the database and viewing all stored records, demonstrating the implementation of basic database operations (CRUD). The project utilizes XAMPP/LAMPP as the local development environment and serves as a practical example of database connectivity, data storage, and web application development.
